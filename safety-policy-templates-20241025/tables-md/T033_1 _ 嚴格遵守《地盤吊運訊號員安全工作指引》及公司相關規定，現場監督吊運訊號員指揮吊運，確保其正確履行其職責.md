# T033 1 / 嚴格遵守《地盤吊運訊號員安全工作指引》及公司相關規定，現場監督吊運訊號員指揮吊運，確保其正確履行其職責

- 来源: `安全管理辦法彙編20241025（無修訂版）.docx`
- 制度上下文: 6.8 地盤負荷物移動機械 > 附件 > 吊運安全守則
- 行列: 8 x 2

| C1 | C2 |
| --- | --- |
| 1 | 嚴格遵守《地盤吊運訊號員安全工作指引》及公司相關規定，現場監督吊運訊號員指揮吊運，確保其正確履行其職責 |
| 2 | 確保吊運工序嚴格依照《吊運安全守則》的要求 |
| 3 | 確保起重機操作員、訊號員、埋碼員等有關人員的資格符合相關要求，起重機械及起重裝置標有清晰安全操作負荷並具備有效檢驗證書 |
| 4 | 確保吊運範圍妥善圍封，沒有其他人和機械在內工作，嚴禁任何人站在懸吊物件之下，且附近沒有潛在危險(如架空電纜、附近正進行其他吊運工作等) |
| 5 | 確保吊運時起重機處於水平狀態，停放的位置足夠堅實、平坦且能承受吊運期間所產生的負荷 |
| 6 | 確保起重機進行吊運時處於良好工作狀態(機件沒明顯破損、損耗、漏油等、安全負荷自動顯示器及支撐腳撐等操作正常) |
| 7 | 確保起重機操作員有清晰無阻的視野，且有吊運信號員在現場指揮 |
| 8 | 如發現任何不安全事項，應向上級及安全管理人員立即匯報並停止吊運 |
