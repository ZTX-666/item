# T055 現委任閣下由即日起擔任本地盤貨車式起重機文件檢查員，負責執行以下事項:

- 来源: `安全管理辦法彙編20241025（無修訂版）.docx`
- 制度上下文: 3. 填妥的表格應交地盤經理簽名確認，副本交地盤安全部存檔備案。 > 貨車式起重機安全吊運守則 > 2. 嚴禁使用葫蘆勾(如事先獲得《使用葫蘆勾工作許可證》者除外)。
- 行列: 5 x 2

| C1 | C2 |
| --- | --- |
| 1 | 在貨車式起重機進入地盤前對如下指定文件進行檢查： 貨車式起重機法定證明書 起重機製造商的操作手冊 起重機維修記錄簿 起重機操作員、訊號員、埋碼員等有關人員的證書 起重裝置的法定證明書 |
| 2 | 檢查貨車式起重機具備全部有效指定文件後，向貨車式起重機發出相應的《貨車式起重機吊運許可證》 |
| 3 | 指示貨車式起重機駛向指定吊運地點 |
| 4 | 通知吊運監督員對貨車式起重機進行吊運檢查 |
| 5 | 對《貨車式起重機吊運許可證》的收發進行管理 |
