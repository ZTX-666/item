# T107 被委任人 承建商代表/地盤經理

- 来源: `安全管理辦法彙編20241025（無修訂版）.docx`
- 制度上下文: 3.加入詳細的密閉空間危險評估表格範本，有關評估報告须於工作完成後保存一年； > 4.在正常大氣壓力下，空氣中氧氣濃度百分比以體積計應不少於百分之十九點五及不多於百分之廿二； > 附件
- 行列: 8 x 2

| C1 | C2 |
| --- | --- |
| 1 | 不應參與其他工作，並曾接受訓練如何與在密閉空間內的工友保持聯絡，須通過雙向通訊設備與密閉空間內的「指定核准工友」至少每2分鐘聯繫1次，以確定密閉空間內的環境沒有異常和在內的「指定核准工友」仍保持清醒 |
| 2 | 在密閉空間外候命，確保工友配戴全身式安全帶的救生繩末端有效連接開腳式救援三腳架或其他起重裝置(工務工程、房屋署、醫管局、港鐵及機管局工程需取得業主代表同意，其他工程按合約要求執行) |
| 3 | 當出現環境轉變和危險(包括水位上升、附近區域出現暴雨、氣温上升或有異味等)，即時指示所有密閉空間工作工友返回地面，並隨時準備施行協助和救援 |
| 4 | 確保密閉空間的通風設備運作正常 |
| 5 | 禁止吸煙及確保在任何密閉空間附近不可生火或使用內燃機(柴油發電機組) |
| 6 | 管理密閉空間入口位置的「出入掛牌制度」，記錄進出密閉空間人員的人數和身份，並須讓密閉空間的負責人士知悉有關記錄，密閉空間工作期間需駐守於入口位置 |
| 7 | 完成密閉空間工作後，檢查所有工友已返回地面，並確保所有密閉空間入口蓋掩已適當地放置 |
| 8 | 遵從指示及建議，出席施工前安全交底、每日開工前安全培訓及專項安全培訓 |
