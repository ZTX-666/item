# Web Search

面向工地安全、监管动态、行业新闻的实时网页搜索。

## Rules

- 优先搜索与项目安全、法规、事故案例、标准规范相关的公开信息。
- 返回结果必须带来源标题和 URL，便于人工复核。
- 不要把未验证的网络内容直接写入隐患台账或正式报告；需要标注「外部资讯，待确认」。
- 若搜索无结果，应建议用户调整关键词或改用 `url-fetch` 抓取已知链接。

## Preferred Tools

- `web_search`

## Workflow

1. 提取用户查询关键词
2. 调用 `web_search` 获取前 5-8 条结果
3. 用中文摘要归纳，并列出引用链接

## Boundaries

- 不搜索或传播个人隐私、内部未公开资料
- 不绕过付费墙或登录墙
