# T116 1 / 為當區安環管理第一責任人，確保當區安環工作能按公司標準、《地盤安環施工管理計劃》及《安全施工計劃》執行，並落實現場監管工作

- 来源: `安全管理辦法彙編20241025（無修訂版）.docx`
- 制度上下文: 六、獎勵標准 > 審批流程 > 附件
- 行列: 8 x 2

| C1 | C2 |
| --- | --- |
| 1 | 為當區安環管理第一責任人，確保當區安環工作能按公司標準、《地盤安環施工管理計劃》及《安全施工計劃》執行，並落實現場監管工作 |
| 2 | 對責任分區進行每日巡查，對於發現的安環違規事項即時安排整改，並跟進相關整改落實情況和拍照記錄 |
| 3 | 若發現的安環違規事項無法有效進行整改，應及時通知地盤安環管理人員並向上級報告 |
| 4 | 負責陪同公司內部、業主、勞工處、環保署或其他政府部門等相關人員到分區進行安環檢查，並講解分區工作情況和即時安排整改 |
| 5 | 對於地盤安環管理人員在日常巡查中發現的違規事項即時安排整改，並在完成後進行拍照記錄和將相片於當日下班前發送至相關安環管理人員存檔 |
| 6 | 按照公司《標準工作程序》的要求，在進行高危工序前提前通知安全管理人員，並按要求做好監管工作 |
| 7 | 分區責任人在早會、定期會議及完成每週安環巡查後召開的專題會議上，講解分區內主要安環管理工作安排、不安全事項和整改跟進情況及下一步工作安排 |
| 8 | 分區責任人應與安環管理人員保持良好溝通，及時反映分區內安環管理情況 |
