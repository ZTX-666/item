# T106 被委任人 承建商代表/地盤經理

- 来源: `安全管理辦法彙編20241025（無修訂版）.docx`
- 制度上下文: 3.加入詳細的密閉空間危險評估表格範本，有關評估報告须於工作完成後保存一年； > 4.在正常大氣壓力下，空氣中氧氣濃度百分比以體積計應不少於百分之十九點五及不多於百分之廿二； > 附件
- 行列: 11 x 2

| C1 | C2 |
| --- | --- |
| 1 | 在未有申請進入密閉空間「許可工作證」的情況下，不可擅自進入密閉空間內 |
| 2 | 根據風險評估報告及安全施工程序要求佩戴及使用合適的安全防護裝備或緊急救援設備，包括全身式安全帶連接獨立救生繩 |
| 3 | 與在密閉空間外駐守的候命人員保持聯絡及聽從指示 |
| 4 | 在密閉空間工作期間，須帶備空氣監測設備及隨身佩戴附有聲光警報功能的動態感應警報器(俗稱懶人鐘) |
| 5 | 依照施工方案、安全施工程序及安全守則施工 |
| 6 | 將安全鍊等放置在設有合適臨時阻截裝置的沙井中，尤其在工作區域的下游 |
| 7 | 避免攪動淤泥，如不可避免時，須經常檢查是否存在有害或有毒氣體 |
| 8 | 留意危險跡象，包括水流量突然增加、有灼熱或有異味的釋出物跡象、空氣監測設備讀數出現變化，以及身體不適等 |
| 9 | 如發現工作環境變得危險或天氣情況惡化時，須立即離開密閉空間 |
| 10 | 遵從「監工或密閉空間工作負責人」要求的安全施工程序執行 |
| 11 | 遵從指示及建議，出席施工前安全交底、每日開工前安全培訓及專項安全培訓 |
