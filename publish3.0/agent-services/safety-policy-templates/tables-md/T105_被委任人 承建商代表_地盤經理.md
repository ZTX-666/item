# T105 被委任人 承建商代表/地盤經理

- 来源: `安全管理辦法彙編20241025（無修訂版）.docx`
- 制度上下文: 3.加入詳細的密閉空間危險評估表格範本，有關評估報告须於工作完成後保存一年； > 4.在正常大氣壓力下，空氣中氧氣濃度百分比以體積計應不少於百分之十九點五及不多於百分之廿二； > 附件
- 行列: 5 x 2

| C1 | C2 |
| --- | --- |
| 1 | 在所有進入密閉空間工作人員離開密閉空間前，不可離開地盤 |
| 2 | 對工作環境和在密閉空間內進行的工作進行風險評估，並就工友進行密閉空間工作時應採取與安全和健康有關的措施提出建議 |
| 3 | 在履行「指定合資格人士」職責期間不得進入密閉空間。如需要進入密閉空間，應事先取得「地盤授權代表」同意，工務工程、房屋署、醫管局、港鐵及機管局工程需同時取得業主代表同意，「指定合資格人士」應持有有效的「指定核准工友」證書。地盤須在「指定合資格人士」進入密閉空間之前準備單獨的風險評估 (並非由打算進入密閉空間的「指定合資格人士」編制)和「許可工作證」，並指定一名候命人員，在其不在密閉空間外期間在密閉空間外駐守，以履行候命人員的職責。「指定合資格人士」進入密閉空間僅限於進行臨時監督、檢查、核對、核實或移交工作，並遵守所有必要的密閉空間安全規定及合約規定，以及針對「指定合資格人士」進入密閉空間的風險評估中發現的危險，採取了所有必要的安全預防措施。「指定合資格人士」在該密閉空間停留期間不得從事「指定核准工友」的工作 。*即如原定的「指定合資格人士」須進入密閉空間(僅限於進行臨時監督、檢查、核對、核實或移交工作)，地盤須委任另一位「指定合資格人士」重新進行風險評估，並重新申請「許可工作證」 |
| 4 | 就相關密閉空間的獨有特徵，向施工團隊提供工作前的講解 |
| 5 | 向「指定核准工友」講解有關安全風險和在進入密閉空間前須採取的安全措施 |
