# 赤瞳灵讯 publish3.0 · UI 与连接状态增强

> 版本：**publish3.0**  
> 在 publish23 基础上增加：主/子页面 UI 统一优化、连接断开状态提示增强。

---

## 1. 视觉与交互

- 主页面保留固定窗体尺寸（统一入口视觉）。
- 其他模块窗口可手动拖动调整尺寸。
- 数据浏览按钮区顺序调整为：导出数据库 / 加载数据库 / 默认数据库 / 刷新 / 搜索。
- 「配置环境」「耀耀工厂」「帮助手册」页面统一适配主风格：颜色、字体、输入框样式一致。

---

## 2. 登录状态与断链提示

- 主页状态灯逻辑增强：
  - 已登录：绿色，显示「已登录」
  - 未登录：红色，显示「未登录」
  - 链路断开：红色，显示「链接断开」
- 登录状态按既有定时机制持续刷新，断链会及时反映到主界面状态条。

---

## 3. 数据库与多账号（继承 publish23）

- 默认数据库根目录：`%UserProfile%\ChitongLingxun`
- 同手机号复用同库，不同手机号新建 `手机号Data Base`
- 修改默认目录时，会尝试迁移已有账号库与 `_unbound`

---

*数据库、云同步与并发控制的详细说明沿用 publish23 / publish22 memory 文档。*
