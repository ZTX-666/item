# T095 1 / 負責對專業工程師提交的臨時支架設計圖則、計算及初步施工方案進行核對或提供改善建議，確定有關設計合理可行，並能應用於地盤施工環境

- 来源: `安全管理辦法彙編20241025（無修訂版）.docx`
- 制度上下文: 四、搭建臨時支架(紅架)安全施工程序(附件11.14) > 五、安全管理 > 附件
- 行列: 4 x 2

| C1 | C2 |
| --- | --- |
| 1 | 負責對專業工程師提交的臨時支架設計圖則、計算及初步施工方案進行核對或提供改善建議，確定有關設計合理可行，並能應用於地盤施工環境 |
| 2 | 檢查建成的臨時支架，以確實它是符合圖紙及規格，並確定該臨時支架是穩固安全地搭建及適宜作預定用途 |
| 3 | 在工程進行期間發現地盤情況改變，以及有關建造、使用或拆卸臨時支架的工序與方法的問題，須向專業工程師及地盤管理層匯報並協調處理。 |
| 4 | 在擺放物料於臨時支架前，檢查確定已完成相關安全措施，並簽發《工作台/臨時支架負載許可證》 |
