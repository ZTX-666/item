# T101 1 / 在合資格的人的直接監督下進行搭建、更改或拆卸金屬棚架

- 来源: `安全管理辦法彙編20241025（無修訂版）.docx`
- 制度上下文: 四、搭建臨時支架(紅架)安全施工程序(附件11.14) > 五、安全管理 > 附件
- 行列: 4 x 2

| C1 | C2 |
| --- | --- |
| 1 | 在合資格的人的直接監督下進行搭建、更改或拆卸金屬棚架 |
| 2 | 正確使用合規格的個人保護裝備如安全帽、安全帶、防墮器及雙尾繩，並把安全帶尾繩扣在穩固點上 |
| 3 | 嚴格遵守勞工處編制的《金屬棚架工作安全守則》及公司《搭建及拆卸臨時支架安全守則》及按照《搭建臨時支架(紅架) 安全施工程序》搭建臨時支架 |
| 4 | 如發現任何不安全事項，或棚架有損毀，應立即向上級及安全管理人員匯報並停止棚架搭建工序 |
