# T099 1 / 嚴格遵守勞工處編制的《金屬棚架工作安全守則》及公司相關規定

- 来源: `安全管理辦法彙編20241025（無修訂版）.docx`
- 制度上下文: 四、搭建臨時支架(紅架)安全施工程序(附件11.14) > 五、安全管理 > 附件
- 行列: 10 x 2

| C1 | C2 |
| --- | --- |
| 1 | 嚴格遵守勞工處編制的《金屬棚架工作安全守則》及公司相關規定 |
| 2 | 理解臨時支架計劃書、設計圖則、規格及施工方案，現場監督整個臨時支架的建造工作,確保依照施工方案及安全守則,嚴格執行各項安全措施, 直至搭建完成 |
| 3 | 安排及直接監督曾受訓練的人(金屬棚架工)搭建及拆卸棚架 |
| 4 | 定期檢查金屬棚架，確保安全可用；如有問題要即時匯報地盤管理人員配合跟進 |
| 5 | 檢查所有曾受訓練的人是否正確使用合規格的個人保護裝備如安全帽、安全帶、防墮器及雙尾繩，並把安全帶尾繩扣在穩固點上 |
| 6 | 在首次使用、每隔十四日及金屬棚架經結構改動或為移動後進行檢查，並填寫法定表格五及張貼在顯眼位置 |
| 7 | 在惡劣天氣後或在棚架改動後，金屬棚架要進行徹底檢查，並填寫法定表格五，及張貼在顯眼位置 |
| 8 | 確保棚架搭建範圍妥善圍封及張貼安全警告告示，沒有未經授權人和機械在內，且附近沒有潛在危險 |
| 9 | 出席有關每日施工前交底及專項安全培訓 |
| 10 | 如發現任何不安全事項，或棚架有損毀，應立即向上級及安全管理人員匯報並停止相關工序 |
