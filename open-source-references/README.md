# 开源参考项目索引

本目录为赤瞳平台功能设计的**参考实现**，不作为运行时依赖。吸收思路后应在 `agent-toolbox` 内实现为内置 tool。

| 子目录 | 参考能力 |
| --- | --- |
| `mcp-construction` | MCP 工具拆分与 schema |
| `HazardLens` | 安全事件闭环、Dashboard |
| `Construction-PPE-Detection` | 多摄像头 PPE 检测 |
| `SafetyVision` | AI 安全报告、RAG |
| `CCTV-Detection-Factory` | 工厂 CCTV 检测 |
| `pdfplumber` / `docstrange` | PDF 解析参考 |
| `sitewatch-ai` / `AI-Safety-Compliance-Officer` | 合规检查流程 |

上传时已排除：`mlruns/`、模型权重、`dist/`、`node_modules/`。
