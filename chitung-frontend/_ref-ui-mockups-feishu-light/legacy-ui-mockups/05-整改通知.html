编号：WARN-20250616-001   |   日期：2025年6月16日

主题：关于 B 区3楼临边围封缺失的安全警告

根据本公司安全管理制度及《工厂及工业经营（安全工作）规例》，现就贵司在启德发展计划基础工程项目中存在的安全隐患，发出本警告信。

一、违规事项

于2025年6月12日，本公司安全主任在巡查中发现：

项 违规描述 级别 地点
1 B区3楼临边护栏于上午被拆除后未恢复，形成约2米宽缺口，仅以警示带临时围封，不符合《临边作业安全管理规定》第5.2条 严重 B区3楼
2 现场观察到有工人穿越警示带进入临边区域，存在坠落风险 严重 B区3楼

二、整改要求

请贵司于2025年6月18日18:00前完成以下整改：

  1. 恢复B区3楼临边护栏，符合《临边作业安全管理规定》要求
  2. 安排专人监督，防止工人穿越警示区域
  3. 提交整改照片及自检报告

三、后续处理

若贵司未能于指定期限内完成整改，或同类违规再次发生，本公司将依据合约条款发出罚款通知，并保留暂停贵司相关作业的权利。
