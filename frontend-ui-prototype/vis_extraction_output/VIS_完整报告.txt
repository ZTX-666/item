【4】品牌规范建议
================================================================================

📌 从 PPT 模板中提取的设计规范:

  1. 字体规范:
     • 标题字体: 黑体 (SimHei)
     • 正文字体: 仿宋 (FangSong)
     • 建议使用字号: 标题 44pt, 正文 18-24pt

  2. 幻灯片尺寸:
     • 16:9 模板: 338.7mm x 190.5mm
     • 4:3 模板: 254.0mm x 190.5mm

  3. 布局类型:
     • 標題投影片 (标题幻灯片)
     • 標題及內容 (标题及内容)
     • 章節標題 (章节标题)

📌 后续步骤:

  由于 PDF 手册为图像格式，建议:
  1. 使用 Adobe Acrobat Pro 进行 OCR 识别
  2. 或手动查看 pdf_pages/ 目录中的图像
  3. 从图像中记录以下信息:
     • 主色调 (品牌主色) 的 RGB/HEX/CMYK 值
     • 辅助色的色值
     • 强调色的色值
     • Logo 最小尺寸和安全区域
     • 禁止使用规范

  4. 预期的主色调 (基于中国建筑品牌):
     • 品牌红: 可能类似 #C4161C 或 #E31837
     • 辅助色: 灰色、白色等
     • 需要 OCR 或手动确认具体色值
